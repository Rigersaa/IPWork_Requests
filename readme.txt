This file includes instructions for installing the IPWorks 2024 JavaScript 
Edition module in Windows and Linux.

This edition is intended for use with Node.js.

************************************************
*         Installation Instructions            *
************************************************

To install the module run the following command:

  npm install ipworks-24.0.tar.gz
  
The module should now be installed along with demos and help files. 
The installation process is the same for Windows and Linux.


************************************************
*        Linux/macOS Trial Licensing           *
************************************************

No special steps are required to activate a trial license. The components may
be used on the machine for 30 days.
  
************************************************
*     Linux/macOS Royalty Free Licensing       *
************************************************

After installing the toolkit no additional steps are required. Applications 
built using the components may be distributed without any additional steps.

************************************************
* Windows Trial Version Licensing Instructions *
************************************************

No special steps are required for the trial versions. Running the trial version
setup will install a license on the machine.

***********************************************
* Windows Full Version Licensing Instructions *
***********************************************

Running the Windows setup on a machine will activate a license, no further 
action is required to license the toolkit. The library may then be deployed
to any other machine without any additional steps.
